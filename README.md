# ancient-prose-notes
## 古风散文与随笔文字合集
中华传统散文承载着古人细腻的生活心境，不同于格律严格的诗词，随笔、游记、小品文行文自由，写景、记事、抒情融为一体，文字清淡雅致，适合日常品读摘抄。本文库整理历代经典小品散文、山水游记、生活随笔、书信短文，兼顾课内古文与课外小众文学篇目，无繁杂广告，纯文字阅读体验。

### 一、文库收录文体分类
1. 山水游记：古人游历山河写下的观景散文，记录各地风光、风土人情；
2. 生活小品：记录日常闲居、品茶观花、独处沉思的短篇随笔；
3. 书信尺牍：古代文人往来书信，文字温柔，兼具叙事与抒情；
4. 说理短文：短小精炼的哲理散文，解读处世、读书、修身之道；
5. 风物杂记：描写草木、时节、器物的写景散文，文字细腻优美。

### 二、文库内容特色
1. 全文搭配通俗白话释义，生僻字词标注读音，零基础也能读懂古文；
2. 拆分段落赏析，梳理文章行文脉络、作者表达的核心情感；
3. 剔除晦涩冗长的古籍长篇，优先选择篇幅适中、适合碎片化阅读的短文；
4. 区分小学、初中、高中适配篇目，学生可根据学习阶段挑选阅读素材。

### 三、适合阅读人群
1. 在校学生，拓展古文阅读积累，提升语文阅读理解能力；
2. 文案爱好者，摘抄古风短句、写景段落用于写作素材；
3. 传统文化爱好者，闲暇品读古典散文，舒缓日常浮躁情绪；
4. 文字创作者，借鉴古人写景抒情的写作手法。
